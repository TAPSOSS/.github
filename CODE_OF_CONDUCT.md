# Code of Conduct

## Our Pledge
We want tapsOSS projects to be welcoming, inclusive, and respectful. By participating in this project, you agree to act professionally and treat others with kindness. Harassment, discrimination, or disruptive behavior will not be tolerated.  

## Reporting Issues
If you encounter any behavior that violates this code, please report it by opening a thread in the **[GitHub discussion boards](https://github.com/orgs/TAPSOSS/discussions)**.  

**Important:** Discussions are public and may not be monitored constantly. While we aim to review issues promptly, there could be delays. Please avoid including sensitive personal information in your report.  

## Enforcement
Maintainers reserve the right to remove, block, or restrict anyone who violates this Code of Conduct. Decisions will be made in line with maintaining a safe and respectful environment for all contributors.  

## Scope
This Code of Conduct applies to all tapsOSS projects, including but not limited to:  
- GitHub repositories  
- Issues and pull requests  
- Discussions  
- Community forums related to the projects
